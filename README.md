# NeoVim Configuration

## Installation

### Linux

```bash
git clone git@github.com:binsim/nvim-config.git ~/.config/nvim
```

### MacOS

```bash
git clone git@github.com:binsim/nvim-config.git ~/.config/nvim
```

### Windows

```bash
git clone git@github.com:binsim/nvim-config.git ~/.config/nvim
```
